# useDebounce

The `useAutoRefresh` automatically calls a callback function on a specified interval.

## Usage

```tsx
const [data, setData] = React.useState();

const getData = () => {
	const newDate = {}; // retrieve updated data
	setData(newData);
};

useAutoRefresh(getData, 5); // refresh every 5 seconds
```
